# Security Policy

## Supported Versions

This project is pre-1.0. Security fixes are handled on the default branch and
released with the next package update.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories or
by opening a private report with enough detail to reproduce the issue. Do not
include real credentials, private URLs, or proprietary documents in public
issues.

linkcheck is local-first and does not upload repository contents. External URL
checking is opt-in with `--check-external`; keep it disabled for private
documentation unless outbound requests are acceptable for that environment.
