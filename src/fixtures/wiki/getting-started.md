# Getting Started

This guide walks you through setting up LinkCheck.

## Installation

Run `npm install` to install dependencies.

```bash
npm install @rogerchappel/linkcheck
```

## Configuration

Create a `linkcheck.config.json` file.

### Advanced Setup

For advanced scenarios, see the [Config Guide](../guides/config.md).

## Usage

Run the scanner:

```bash
linkcheck scan ./docs
```
