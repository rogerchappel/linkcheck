# API Reference

## Overview

The LinkCheck API provides link scanning capabilities.

## Endpoints

### scan

Scans files for broken links.

Parameters:
- `path`: File or directory to scan
- `checkExternal`: Whether to check external URLs

### report

Returns the last scan report.
